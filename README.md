# ViLT for Autonomous Driving

## Project Overview

This project fine-tunes a Vision-Language Model (**ViLT**) on the **Udacity Self-Driving Car Dataset** for **image–text understanding** in autonomous driving scenarios.

## Dataset

* **Dataset:** Udacity Self-Driving Car Dataset
* Road scene images from autonomous driving environments

## Model

* **Model:** ViLT (Vision-Language Transformer)
* **Task:** Image Captioning / Scene Understanding

## Platform

* Google Colab

## Features

* Fine-tuned ViLT model
* Image-to-text understanding
* Predicted scene captions
* Visual output generation

## Sample Output

Example caption:

> “A city street with traffic and vehicles nearby.”

## Technologies Used

* Python
* PyTorch
* Hugging Face Transformers
* Google Colab

## Future Improvements

* Train on larger datasets like BDD100K
* Compare with BLIP and LLaVA
* Improve caption accuracy
